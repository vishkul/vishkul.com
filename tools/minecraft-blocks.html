// pixel to blocks · experiments

A Minecraft builder
that reads your image.

Drop or paste an image and it gets colour-matched, cell by cell, to real Minecraft blocks. Set your version and how you play, then get the blocks: a zoomable map you can read square by square, exact counts, real block textures, and a list you can copy straight into your build. Verified colours, no AI, no API.

Flat build — pixel art, walls, floors, facades: anything one block thick. You get a block map and exact counts, because every block is one cell in a flat grid.

limits the palette to blocks that existed in that version. concrete & glazed arrive in 1.12; nether blocks 1.16; deepslate 1.17.
build in one material, or mix for the closest colour match.
This is a reference, not a build plan. A single photo only shows one or two faces of a 3D thing, so there's no honest way to know its full structure or give a total block count.
