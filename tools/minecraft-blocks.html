// image to blocks · experiments

Turn any image into
Minecraft blocks.

Drop in an image and it gets colour-matched, pixel by pixel, against a palette of real Minecraft blocks with verified colours. Pick whether you're in Creative or Survival, set how big the build is, and get your blocks. No AI, no API, no guessing on colours: just maths against a hand-checked block list.

Flat build: for pixel art, walls, floors, banners, facades, anything one block thick. You get a block-by-block map and exact counts, because every block is one cell in a flat grid.

Heads up, this is a reference, not a build plan. A single photo only shows one or two faces of a 3D thing, so there's no honest way to know its full structure or give you a total block count. What you get below is the matched block palette, the real blocks whose colours best fit the image, so you can build it yourself in-game and know exactly which blocks to reach for. The structure is yours to build; these are your materials.
